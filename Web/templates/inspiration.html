Inspiration Websites

Website One: Ammunition Group

The main inspiration I got from this website was the design of the jumbotron. I found this was a striking and impactful image for the user to see when they open the page. It also has itâ€™s navigation at the top of the page. This is very common and I chose to also incorporate this feature into my website.

This website also has a flat style which is very contemporary. The buttons and links all have no shading or 3D styling, instead they are sharp lines and no rounded corners and simple colours. This gives a flat modern appearance. When adding buttons to my page I emulated this look for the axis buttons.

Website Two: Template monster apple theme

The colour choices on this website were quite useful. It is recommended that if you do not have a logo or brand then you should choose only a few colours for your website. The website uses 3 colours and then mainly neutral colours throughout the rest of the website. These colours being repeated gives the whole website a level of cohesion and ties all the elements together. It also helps to create a sense of branding for the whole website on all pages and connect all the components.

I was not only inspired by the choice in limited colours but also the colours they used. The purple and pink created a visual difference but also complemented each other, that is why I chose to use variants of those colours on my website. While the website used a green accent colour, I chose not to use it for my own website because I was not confident in my ability to use it effectively like they have.

Website Three: Interstate Team

The Interstate Team website has a clean modern look to it. There is a minimalistic style and all the objects on screen are kept in a grid format. The grid look is very contemporary and keeps all the elements organised for easy comprehension by the user.

The menu in the top left corner was a hamburger menu. This is a bit outdated for most modern websites, so I decided not to incorporate that into my website and stick to the more common top navigation menu.
